# resign-claude

퇴사를 기념하는 3단계 웹 앱. 사직서를 쓰고, 감사패를 받고, 퇴직금을 확인하세요.

## 화면 구성

| 단계 | 페이지 | 내용 |
|------|--------|------|
| 1 | 사직서 | 회사명, 성명, 월 급여, 재직 기간, 사직 사유 입력 |
| 2 | 감사패 | 재직 기간을 담은 감사패 생성 |
| 3 | 퇴직금 영수증 | 퇴직금 계산 결과 + 돈 비 애니메이션 |

## 퇴직금 계산 방식

> 퇴직금 = 1일 평균임금 × 30일 × (재직일수 / 365)
>
> 1일 평균임금 = 월 급여 ÷ 30

법정 퇴직금 산정 기준(근로자퇴직급여 보장법)을 단순화한 공식으로, 참고용입니다.

## 기술 스택

- **React 19** + **TypeScript**
- **Vite 8**

## 실행

```bash
npm install
npm run dev
```
